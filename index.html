Interests

I'm interested in the design and implementation of programming languages. My current research focuses on simple Just-In-Time (JIT) compilation techniques and optimizations allowing to efficiently implement dynamic programming languages. For my Ph.D., I implement these techniques and optimizations in a JIT compiler for the Scheme programming language called LC.
I'm also interested in real time rendering techniques. I built several tools to explore this topic.
